Документация API

Работа с выражениями

Создание нового выражения

POST /expressions

Проверяет ключ идемпотентности и создаёт новую запись с выраженим в базе данных и возвращает её идентификатор

Тело запроса

{ "expression": "2+2*2", "idempotencyKey": "UUID_KEY" }

Тело ответа

{ "id": 1 }

Получение информации и о выражении

GET /expression/:id

Получает информацию о выражении из базы данных, возвращает его статус и результат

Тело ответа

{ "id": 1, "expression": "2+2*2", "createdAt": "2024-02-16T19:33:27.898659Z", "finishedAt": "2024-02-16T19:33:27.898659Z", "status": 3, "result": 6 }

Получение списка всех выражений

GET /expressions

Возвращает информацию о всех выражениях, записанных в базе данных

Тело ответа

[ { "id": 1, "expression": "2+2*2", "createdAt": "2024-02-16T19:33:27.898659Z", "finishedAt": "2024-02-16T19:33:27.898659Z", "status": 3, "result": 6 } ]

Работа с операторами

Сохранения нового времени выполнения для каждого оператора

POST /operators

Сохраняет новое время выполнения для каждого оператора

Тело запроса

[ { "operationType": 0, "durationMS": 100 }, { "operationType": 1, "durationMS": 200 }, { "operationType": 2, "durationMS": 300 }, { "operationType": 3, "durationMS": 400 } ]

Получение информации обо всех операторах

GET /operators

Возвращает тип каждого оператора и время его выполнения

Тело ответа

[ { "operationType": 0, "durationMS": 100 }, { "operationType": 1, "durationMS": 200 }, { "operationType": 2, "durationMS": 300 }, { "operationType": 3, "durationMS": 400 } ]

Работа с агентами

Добавление нового агента

POST /worker

Добавляет информацию о новом агенте в базу данных

Тело запроса

{ "id": 1, "url": "http://localhost:8001", "executors": 10 }

Получение информации обо всех огентах

GET /workers

Возвращает информацию обо всех доступных агентах из базы данных

Тело ответа

[ { "id": 1, "url": "http://daemon1:8001", "executors": 1, "lastModified": "2024-02-18T15:43:22.456728Z" }, { "id": 2, "url": "http://daemon2:8002", "executors": 5, "lastModified": "2024-02-18T15:43:22.586835Z" } ]

Получение задач, выполняемых агентом

GET /worker/:id/tasks

Возвращает список задач, выполняемых агентом

Тело ответа

[ { "leftResult": 2, "operationType": 0, "rightResult": 6, "status": 2 }, { "leftResult": 2, "operationType": 2, "rightResult": 3, "status": 2 }, { "leftResult": 2, "operationType": 0, "rightResult": 6, "status": 2 } ]

Работа с задачами

Задача - простое арифметическое выражение из одной операции, которое может посчитать агент. Пути из этой группы используются только внутри приложения агентами

Начало работы над задачей

POST /task/:id/status

Ставит статус для задачи "выполняется". Вызывается, когда агент начинает работу над задачей

Получение результата задачи

POST /task/:id/result

Записывает результат и статус для задачи

Тело запроса

{ "result": 10 }
